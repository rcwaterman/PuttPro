PuttPro

Setup:
1. Run python app.py on your PC (same WiFi network).
2. Tap Find Server Automatically — works instantly on iOS and modern Android via mDNS (puttpro.local).
3. Or scan the QR code from the server startup output.
4. Or paste the Mobile URL manually and tap Connect.

Certificate warning? Scan the CA QR from startup output and install PuttPro-CA.crt on this device first.
